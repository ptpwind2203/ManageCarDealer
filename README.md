# ManageCarDealer

Hệ thống quản lý đại lý xe ô tô.

## Công nghệ sử dụng

- Frontend: Vue.js
- Công cụ phát triển Frontend: Visual Studio Code
- Backend: Spring Boot
- Công cụ phát triển Backend: IntelliJ IDEA
- Database: MySQL
- Công cụ quản lý CSDL: phpMyAdmin
  
## Chức năng

- Quản lý xe
- Quản lý chi tiết xe
- Quản lý màu sắc (đang phát triển)
- Quản lý phiên bản (đang phát triển)
- Quản lý tài khoản (đang phát triển)
- Quản lý hóa đơn (đang phát triển)
- Trong quá trình phát triển có thể có thêm chức năng khác....
